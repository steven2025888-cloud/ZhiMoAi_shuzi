【快速开始】

Python 代码示例：
```python
import whisper

# 加载模型
model = whisper.load_model("base")

# 转录音频
result = model.transcribe("audio.mp3", language="zh")

# 获取文本
print(result["text"])
```

命令行使用：
```bash
IndexTTS2-SonicVale\installer_files\env\Scripts\whisper.exe audio.mp3 --language Chinese
```


【重要提示】

1. FFmpeg 依赖
   Whisper 需要 FFmpeg 处理音频文件
   织梦AI 已包含 FFmpeg：LatentSync/ffmpeg-7.1/bin/ffmpeg.exe
   
   如果遇到 FFmpeg 错误，请将 FFmpeg 添加到系统 PATH

2. 首次使用
   首次使用某个模型时会自动下载
   模型保存在：C:\Users\用户名\.cache\whisper\
   
3. GPU 加速
   如果有 NVIDIA 显卡，Whisper 会自动使用 GPU 加速
   CPU 模式下请设置 fp16=False
